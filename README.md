### En:
#### Task 1
What You Need to Do
Task 1
Create a small web application for displaying a delivery list that a courier could use in their work.
To create a delivery card, define a class called Delivery, whose constructor takes three arguments:
- the customer’s name,
- the address, and
- the distance.
Example of initialization:
new Delivery("Olga", "12 Vymyslov St.", 8);


Since this is a list of deliveries, you should store the class instances in an array, which can be easily processed using a loop.
Example array:
const deliveryArr = [
  new Delivery("Olga", "12 Vymyslov St.", 8),
  new Delivery("Dmitry", "7 Zadachnaya St.", 3),
  new Delivery("Olya", "43 Tkachyov St.", 11)
];

### Ru:
#### Задача 1
Что нужно сделать
Задание 1
Создайте небольшое веб-приложение для отображения списка доставок, которым мог бы пользоваться курьер в своей работе. Для создания карточки доставки создайте класс Delivery, конструктор которого принимает три аргумента: имя покупателя, адрес и расстояние. Инициализация должна выглядеть следующим образом:

Пример инициализации: 

new Delivery("Ольга", "ул. Вымыслов, д. 12", 8);
Так как это список доставок, то для хранения экземпляров классов нужно использовать массив, который удобно обрабатывать циклом.

Пример массива:

const deliveryArr = [
  new Delivery("Ольга", "ул. Вымыслов, д. 12", 8),
  new Delivery("Дмитрий", "ул. Задачная, д. 7", 3),
  new Delivery("Оля", "ул. Ткачей, д. 43", 11)
];
